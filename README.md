# React Class Binding

=> CLass Binding is a technique used to configure CSS classes to elements dynamically.
=> JSX can't use "class" attribute to bind CSS class.
=> JSX requires "className" property to binding a CSS class.
=> Class Name is defined in a string format.
Syntax: <h1 className="text-primary bg-danger">

=> Multiple classes a separated with blank space.
=> You can bind a CSS class dynamically using data binding expression.
